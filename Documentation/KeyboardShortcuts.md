# Keyboard Shortcuts

## Playback

| Keystroke | Action |
|-----------|--------|
| Space | Play/Pause |
| ⌘E | Next song |
| ⌘L | Like current song |
| ⌘D | Dislike current song (skips to next) |
| ⌘T | Tired of song (skips to next) |
| ⌘↑ | Increase volume |
| ⌘↓ | Decrease volume |

## Stations

| Keystroke | Action |
|-----------|--------|
| ⌘N | Create new station |
| ⌘R | Reload stations list |

## Window

| Keystroke | Action |
|-----------|--------|
| ⌘, | Preferences |
| ⌘W | Close window |
| ⌘M | Minimize window |
| ⌘H | Hide Hermes |
| ⌘Q | Quit Hermes |

## Media Keys

Hermes responds to standard media keys on Apple keyboards:

- **Play/Pause** (F8 or ▶️⏸️)
- **Next Track** (F9 or ⏭️)

Media key support can be toggled in Preferences → General.
